# 第 4 章 · LLM API 调用

会用 API 是 Agent 开发的基础。不管你的 Agent 架构多复杂，最终都要通过 API 与 LLM 交互。这一章从第一次调用开始，逐步覆盖到生产环境需要的全部能力。

## 本章三层结构

### 初级：第一次 API 调用

掌握 Anthropic Claude 和 OpenAI 两套 API 的使用方法，理解请求响应结构，实现多轮对话。学完能独立调用 LLM API 完成各种任务。

### 中级：流式输出与多模态

实现流式（Streaming）输出的前后端完整链路，掌握图片和 PDF 的多模态输入，学会设计统一的 LLM 适配层。学完能构建有流式打字效果的 AI 对话应用。

### 高级：错误处理与生产化

掌握指数退避重试、速率控制、Token 截断等生产级策略，实现一个健壮的 LLM 客户端。学完能在生产环境中安全可靠地使用 LLM API。

## 学完能做到

| 层级 | 学完你能 |
|------|---------|
| 初级 | 调用 Claude/OpenAI API，实现多轮对话，理解 Token 计费 |
| 中级 | 实现流式输出，处理图片/PDF 输入，设计多模型适配层 |
| 高级 | 构建带重试、限流、降级的生产级 LLM 客户端 |

## 前置要求

- 已注册 Anthropic 或 OpenAI 账号并获取 API Key
- 掌握 Python 基础语法和 async/await
- 了解 HTTP 协议基础
