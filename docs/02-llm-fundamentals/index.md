# 第 2 章 · LLM 原理

大语言模型（LLM）是 Agent 的"大脑"。你不需要能训练一个模型，但你需要理解它是怎么工作的 -- 就像你不需要造发动机，但开车得知道油门和刹车的原理。

## 本章三层结构

### 初级：LLM 是什么

用通俗的语言解释 LLM 的核心概念。什么是 Token，主流模型有哪些，为什么会"幻觉"。学完能在团队里聊 AI 不露怯。

### 中级：Transformer 怎么工作

深入注意力机制的直觉理解，搞清楚 Q/K/V 到底在干什么，理解 Temperature 和 Top-p 如何影响输出。学完能解释模型参数调优的原理。

### 高级：底层优化

用代码实现注意力计算，理解 KV Cache、Flash Attention 等工程优化，了解模型量化。学完能理解推理引擎在做什么。

## 学完能做到

| 层级 | 学完你能 |
|------|---------|
| 初级 | 理解 AI 的基本工作方式，对 Token 和模型选择有清晰认识 |
| 中级 | 理解 Transformer 架构，能解释模型参数对输出的影响 |
| 高级 | 理解底层优化原理，能评估推理性能瓶颈 |

## 前置要求

- 初级：无前置要求
- 中级：基础线性代数概念（矩阵乘法）
- 高级：能写 Python，了解 numpy 基础
